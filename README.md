# Armstrong

**Uppgift**
Ett Armstrong-tal är ett tal som är lika med summan av kubiken av dess siffror.

Till exempel 0, 1, 153, 370, 371, 407 etc.

För att förstå varför 153 är ett Armstrong-tal

153 = (1*1*1) + (5*5*5) + (3*3*3)

Där:

(1*1*1) =   1
(5*5*5) = 125
(3*3*3) =  27

1 + 125 + 27 = 153
